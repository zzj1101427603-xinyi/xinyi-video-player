# 馨逸超可爱 - 后台视频播放器

专为iQOO Z5设计的后台视频播放器，支持锁屏和后台播放，适合网课学习。

## ✨ 功能特点
- 🔒 **锁屏后台播放**：锁屏或切换应用时音频继续播放
- 🎵 **通知栏控制**：在通知栏快速控制播放/暂停
- ⚡ **倍速播放**：支持0.5x-2.0x倍速调整
- 📱 **iQOO系统优化**：针对OriginOS/Funtouch OS优化
- 💖 **个性化界面**：专属欢迎标语"为了馨逸，加倍努力！"
- 🔄 **自动扫描**：自动扫描手机中的视频文件

## 📱 启动画面
每次启动APP会显示：
馨逸超可爱
为了馨逸，加倍努力！
## 🔧 使用方法
1. 安装APK到iQOO Z5手机
2. 首次启动允许所有权限请求
3. APP会自动扫描手机中的视频文件
4. 选择要播放的视频
5. 开始播放后即可锁屏或切换到其他应用
6. 音频会在后台继续播放

## ⚙️ 权限说明
| 权限 | 用途 |
|------|------|
| 存储权限 | 访问手机中的视频文件 |
| 通知权限 | 显示播放控制通知 |
| 后台权限 | 允许在后台继续播放音频 |

## 🚀 技术栈
- React Native
- Expo AV（视频播放）
- Expo Notifications（通知）
- Expo Media Library（媒体文件访问）
